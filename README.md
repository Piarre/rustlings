# :crab: Rustlings

Small exercises to get you used to reading and writing Rust code!

From [rust-lang/rustlings](https://github.com/rust-lang/rustlings)

## :rocket: Running the exercises
Requirements: docker

Running

In VS Code : `ctrl+shift+p` or `F1` -> `>Dev Containers: Rebuildand Reopen in Container` -> `rustlings`
